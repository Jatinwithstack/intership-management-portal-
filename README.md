# Internship Management Portal

A lightweight, high-performance, and fully accessible multi-page web application built entirely using **Pure Semantic HTML5**, violating zero layout design guidelines (No layout `<div>` tags utilized).

## 🚀 Project Features
- **Data Dashboard:** Real-time progress metrics using native `<progress>` and `<meter>` elements.
- **Massive Forms:** Over 20+ strictly validated inputs with document upload capabilities.
- **Structured Records:** Advanced tabular tracking with data optimization using `colspan`.
- **Rich Multimedia:** Embedded contextual routes using native `<audio>`, `<video>`, and layout `<iframe>`.

## 📁 Repository Structure & Branching Strategy
- Multi-branch environment managed across `feature/*`, `bugfix/*`, and `release/*` workflows.
- Strictly monitored commit logs to mimic standard industrial agile development pipelines.
